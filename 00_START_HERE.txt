═══════════════════════════════════════════════════════════════
  🆘 QUICK TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

Problem: Backend won't start
  → Make sure PostgreSQL is running
  → Check .env credentials
  → See TROUBLESHOOTING.md

Problem: Port 5000 in use
  → Kill process: taskkill /F /IM node.exe
  → Or use different port in .env

Problem: Database connection failed
  → Check PostgreSQL is running
  → Verify .env credentials
  → Create database: psql -U postgres -c "CREATE DATABASE pharmacy_ai;"

Problem: No medicines in database
  → Run: cd BackEnd && node seed-medicines.js

Problem: Chat returns error
  → Check backend console for error
  → Run: node test-medical-report.js
  → See TROUBLESHOOTING.md for help
